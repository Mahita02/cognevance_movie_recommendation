# Movie Recommendation System Report

## Objective
To build a movie recommendation system using collaborative filtering techniques.

## Dataset
MovieLens Latest Small Dataset

## Steps Performed

1. Dataset Collection
2. Data Cleaning
3. Data Analysis
4. Visualization
5. Recommendation System Development
6. Output Generation

## Results

Top Recommended Movies:
- Heat (1995)
- Pulp Fiction (1994)
- Jurassic Park (1993)
- Star Wars Episode IV (1977)
- The Usual Suspects (1995)

## Conclusion

The recommendation system successfully suggests movies based on user similarity using collaborative filtering.
