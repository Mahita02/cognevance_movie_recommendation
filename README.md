# Movie Recommendation System

## Project Overview
This project is an AI-based Movie Recommendation System developed using Python and Machine Learning techniques.

## Features
- Data Cleaning and Preprocessing
- Movie Ratings Analysis
- Top Rated Movies Visualization
- Collaborative Filtering Recommendation System
- Movie Recommendations for Users

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset
MovieLens Latest Small Dataset

## Project Structure

dataset/
screenshots/
report/
app.py
requirements.txt
README.md

## Output
- Top 10 Most Rated Movies Graph
- Recommended Movies List
- CSV Report of Recommendations
